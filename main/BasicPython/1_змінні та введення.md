# Курс: Змінні та введення в Python

Цей розділ знайомить початківців з базовими змінними, типами даних та введенням даних. Кожна тема містить пояснення та покрокові інструкції для самостійного написання програм.

## 1. Що таке змінні?

Змінні зберігають дані, як коробки з етикетками. Python сам визначає тип: рядок для тексту, int для цілих чисел, float для чисел з комою, bool для істинно/хибно.

**Як використовувати:**

1. Придумайте назву змінної (наприклад, `name`, `age`)
2. Запишіть `назва = значення`
3. Використовуйте назву для доступу до значення

**Практика:** Створіть три змінні для свого імені, віку та зросту.

## 2. Виведення даних на екран

Функція `print()` показує дані користувачу.

**Як використовувати:**

1. Напишіть `print(що_показати)`
2. Можна показати текст у лапках, змінні або комбінацію через кому
3. Для гарного формату використовуйте f-рядки: `f"текст {змінна}"`

**Практика:** Виведіть привітання з вашим ім'ям та віком трьома способами.

## 3. Введення даних від користувача

Функція `input()` запитує текст у користувача і повертає рядок.

**Важливо:** Навіть числа вводяться як текст! Перетворюйте: `int()` для цілих, `float()` для дробових.

**Як використовувати:**

1. Напишіть `змінна = input("питання: ")`
2. Для чисел: `число = float(input("питання: "))`
3. Використовуйте введене значення в обчисленнях

## Задача 1: Простий калькулятор

**Мета:** Прийняти два числа, показати суму, різницю, добуток, частку.

### Покроковий план:

1. Запросіть перше число через input(), перетворіть у float, збережіть у змінну
2. Запросіть друге число аналогічно
3. Створіть 4 змінні для результатів: сума (+), різниця (-), добуток (*), частка (/)
4. Виведіть всі 4 результати гарним форматом
5. Протестуйте з різними числами

## Задача 2: Привітання

**Мета:** Запитати ім'я, вивести "Привіт, [ім'я]!"

### Покроковий план:

1. Запросіть ім'я через input()
2. Виведіть привітання трьома способами:
    - Через кому в print()
    - Через знак + для з'єднання рядків
    - Через f-рядок (найкращий варіант)

## Задача 3: Цельсій → Фаренгейт

**Мета:** Перетворити температуру за формулою F = C * 9/5 + 32

### Покроковий план:

1. Запросіть температуру Цельсія як float
2. Помножте на 9/5 і додайте 32, результат збережіть у змінну
3. Виведіть результат з позначками градусів

**Підказка:** 9/5 в Python = 1.8. Тестуйте з 0°C (32°F) і 100°C (212°F).
